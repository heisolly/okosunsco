# Okosun, Okosun & Partners

Strategic Legal Counsel and Advocacy.

## Overview

Official digital platform for Okosun, Okosun & Partners, a leading Nigerian law firm specializing in high-stakes litigation, corporate law, and property disputes.

## Technologies

- **Frontend**: React 19, TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React

## Development

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
